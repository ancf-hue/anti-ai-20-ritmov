## 📱 О проекте

**«Анти ИИ 20 ритмов»** — это мобильное веб-приложение для развития чувства ритма. Включает **20 различных ритмов** от простых до сложных, визуализацию долей и систему оценки точности.

Приложение работает полностью в браузере, не требует установки, поддерживает установку на экран телефона как PWA и работает офлайн после первого открытия.

## 🎮 Как играть

1. **Выберите ритм** из выпадающего списка
2. **Нажмите «ПРОИГРАТЬ РИТМ»** — метроном проиграет ритм
3. **Слушайте и запоминайте** — визуальные кружочки показывают доли
4. **Отбивайте ритм** — нажимайте на барабан или клавишу **Пробел**
5. **Получайте оценку** — после завершения ритма покажется точность
6. **Улучшайте результат** — старайтесь достичь 100%

### Шкала точности

| Отклонение | Оценка |
|------------|--------|
| < 30 мс | 🎯 ИДЕАЛЬНО! |
| < 150 мс | ⚡ ПОПАЛ |
| > 150 мс | 😅 МИМО |

### Итоговая оценка

| Точность | Результат |
|----------|-----------|
| ≥ 80% | ⭐ ОТЛИЧНО |
| ≥ 50% | ✔️ ХОРОШО |
| < 50% | ❌ МАЛО |
